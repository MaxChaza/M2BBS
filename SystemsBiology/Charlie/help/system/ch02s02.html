Show

In this menu you find items for showing the Petri net in the program Snoopy (if it is installed) or you can view some debug information about the loaded net.

Show in Snoopy

By clicking on this menu item, Charlie tries to open Snoopy with the currently loaded Petri net file.

If the item is clicked for the first time, Charlie asks for setting up the path to the Snoopy executable. By clicking on "yes" one can choose the location of the executable via a file chooser. Otherwise or if no executable file is selected Charlie does nothing.

Debug Petri Net

After clicking this item, a new windows opens, showing some properties of the loaded Petri net like the ids of the places and transitions. This menu item is intended to be used by programmers of Charlie and programmers of Charlie-Plugins.
