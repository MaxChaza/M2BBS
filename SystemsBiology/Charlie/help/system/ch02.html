In this chapter you can find out more about the different menus.

File

In this menu you find items for opening and reloading files, restoring sessions, opening recently opened files and quiting Charlie.

Open

With this menu item you can open Petri net files. A file dialog will pop up so that you can select a file containing the information about the net.

Currently Charlie can open the following file formats: spept, apnn, spmnet, sptpt, spm, spstochpn, spped, spcontped.

Reload

With this menu item one can reload the currently opened Petri net file. This is useful if one has edited the opened file and wants to analyse the changes.

Recent Files

In this list you can find files that have recently been opened. By clicking on one of the file in the list, one can directly open the file.

After restarting Charlie it is checked if the files still exist. All files that no longer exist are removed from the list.
