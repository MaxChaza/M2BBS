GNA Plugin: NuSMV Implicit

This plug-in allows GNA to communicate with the model checker NuSMV, either through the export of the state transition graph to a text file, or through the model-checker web server at INRIA. In the latter case, the results of the verification of a temporal-logic query are displayed in the graphical user interface of GNA.

The state transition graph is represented implicitly, that is, as a successor function in the SMV programming language. This plug-in supports the verification of properties specified in CTL.

Developers: Grégory Batt, Bruno Besson, Hidde de Jong, Pedro Monteiro, Michel Page
