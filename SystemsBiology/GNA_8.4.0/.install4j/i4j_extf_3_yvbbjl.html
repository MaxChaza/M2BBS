GNA is developed in collaboration with the Ibis Group at INRIA Grenoble-Rhone-Alpes and uses the following open-source software: JGraph, SAT4J, and CUP. The model-checker web server at INRIA runs NuSMV.

You can find documentation about GNA:

  • in the GNA help menu, upper right hand corner (Help)
  • on your computer, in the "docs" directory in the GNA file you just installed
  • on the Genostar web site at www.genostar.com and on the Ibis web site at ibis.inrialpes.fr

If you have problems with GNA, send an email to support@genostar.com or Hidde.de-Jong@inria.fr describing the problem. Technical support will respond as promptly as possible.

For GNA updates or patches, check the Genostar and Ibis web sites.
